Recent Projects

inlike

(Live / Repo)

inlike is a lightning-dating web app that allows users rapidly "like" other users. Users are alerted if their likes are reciprocated, allowing them to communicate the other user. As users like or dislike others, more users will load and populate the screen. inlike's features include:

  • Single-page app utilizing client-side Backbone.js and Ruby on Rails backend.
  • OmniAuth Facebook login and user photo upload with Facebook Graph API.
  • Heavy jQuery interaction.
  • File upload with Amazon Web Services (AWS) and Ink File Picker.
  • Backbone Relational, including many-to-many relationships.
  • Utilizes a bin-sorting algorithm with Packery.js, maximizing the number of profile pages that can fit onto the user's screen.
Versa

(Live / Repo)

Versa allows users to dynamically annotate and view annotations for song lyrics. When users highlight text, an annotate button appears that will allow them to attach an annotation to that line. Users can upload songs, view others' annotations, and follow other users. Versa's features include:

  • Single-page app utilizing Backbone.js on the frontend and Ruby on Rails backend.
  • Backbone Relational, including many-to-many relationships.
  • Custom-built jQuery dynamic text annotation.
  • Dynamic search with Solr (server-side) and Select2 (client-side).
